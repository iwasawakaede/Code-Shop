# Code-Shop
##　今回作成したのは、HTML/CSS/javascript を使用した便利な小、中規模の機能を検索、販売するサイトです。
##　対応しているレスポンシブはざっくりと PC、スマホ で画面の表示を切り替えています。

##　ヘッダー部分の入力した検索ワードで当てはまるコードがヒットするようにしています。
##　購入から決済以降の処理は note へのリンクで全て行うようにしています。

# 使用限度
##　HMTL
##　CSS
##　javascript
##　PHP
